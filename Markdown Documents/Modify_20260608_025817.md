# Modify Summary — 2026-06-08 02:58 UTC+8

## 概述

本次修改实现了 ImGui DLL 通过文件桥接（`imgui_bridge.json`）与 GameMaker GML 进行双向通信，覆盖 Gameplay 分类的全部设置项。

---

## 修改清单

### C++ DLL 文件

#### 1. `extshell.cpp` → `execute_shell_simple_ext_x64.dll`

**功能：**
- 保持 4 参数签名，兼容原版 GameMaker Extension
- `@@internal:` 前缀触发内部 DLL 加载/调用
- 支持 `DL:` / `DLLCALL:` / `@` 等多种前缀（动态跳过前缀到 `:`）
- `internalDispatch` 解析 `dllPath|funcName|arg0|arg1` 格式
- 自动检测 float 参数（`.` 或函数名含 `Float`）
- 调用后不 `FreeLibrary`（ImGui 持有持久状态）

**导出函数：**
| 函数 | 签名 |
|------|------|
| `execute_shell_simple_raw` | `BOOL(LPCCH, LPCCH, LPCCH, int)` |
| `ExecuteExternalLibrary` | `BOOL WINAPI(LPCWSTR, LPCSTR)` |

#### 2. `imgui_dll.cpp` → `imgui.dll`

**功能：**
- 无边框窗口（`WS_POPUP`，400x570）
- 始终置顶（`HWND_TOPMOST`）
- 顶部 20px 区域可拖拽
- F1 全局切换显示/隐藏（`GetAsyncKeyState` 100ms 轮询）
- WARP 软件渲染（避免与 GameMaker D3D11 设备冲突）
- 文件桥接：`%LOCALAPPDATA%\VIVIDSTASIS\imgui_bridge.json`

**导出函数：**
| 函数 | 签名 |
|------|------|
| `ImGui_Show` | `void(void)` — 启动渲染线程 |
| `ImGui_Hide` | `void(void)` — 停止窗口 |

## 数据流

```
GameMaker (o_imgui_sync Step)          imgui.dll (Thread)
       │                                       │
       │  game → Init: WriteData()              │
       │  {"note_speed":5, ...}                 │
       │──────────────────────────────────────►│
       │  imgui_bridge.json                    │
       │                                       │  每帧 ReadData()
       │                                       │  UI 滑块 → 变化 → WriteData()
       │                                       │
       │  每10帧: ReadData()                    │
       │  get_json_val(raw, "note_speed")       │
       │◄──────────────────────────────────────│
       │  imgui_bridge.json                    │
       │                                       │
       │  if (_val != global.op_note_speed)     │
       │    global.op_note_speed = _val         │
       │    mod_scrollspeed 更新                │
```
